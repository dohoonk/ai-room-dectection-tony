# Demo Video Production Checklist

## Pre-Recording Setup

### Environment
- [ ] Backend server running on port 8000
- [ ] Frontend running on port 3000
- [ ] Browser window maximized and clean
- [ ] Test data files ready:
  - [ ] `20_connected_rooms_floorplan.json`
  - [ ] `50_rooms_floorplan.json`
  - [ ] `complex_floorplan.json`

### Screen Recording Software
- [ ] Screen recorder installed and configured
- [ ] Recording area set to full screen or browser window
- [ ] Audio input configured (for narration)
- [ ] Cursor highlighting enabled (if available)
- [ ] Recording quality set to HD (1920x1080 minimum)

### Application State
- [ ] Application tested and working
- [ ] No console errors
- [ ] All features functional (detection, rename, delete)
- [ ] Sample data validated

## Post-Recording Review

### Technical Quality
- [ ] Video resolution is HD (1920x1080+)
- [ ] Frame rate is smooth (30 FPS+)
- [ ] Audio is clear and audible
- [ ] No screen flickering or glitches
- [ ] Cursor movements are smooth

### Content Quality
- [ ] All key features demonstrated
- [ ] Workflow is clear and logical
- [ ] Time savings are evident
- [ ] Multi-room detection shown
- [ ] Interaction features demonstrated
- [ ] Metrics are visible and explained

### Professional Appearance
- [ ] Clean, professional presentation
- [ ] No unnecessary UI elements visible
- [ ] Smooth transitions between scenes
- [ ] Appropriate pacing (not too fast/slow)
- [ ] Clear visual focus on key elements

## Distribution Formats

### Primary Format
- **File**: `room-detection-ai-demo.mp4`
- **Resolution**: 1920x1080
- **Codec**: H.264
- **Bitrate**: 5-10 Mbps

### Alternative Formats (if needed)
- Web-optimized version (lower bitrate)
- Mobile-friendly version (vertical format)
- GIF version for quick preview
